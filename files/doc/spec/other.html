Interrupts

Invoke an interrupt with the * operator.
eg: *0xbeef

Syscalls

Syscalls are a special case of an interrupt, accepting arguments and return types.
Adressed by name.
eg: *write 1, "Hello, World!", 13

Builtin

Envoked with the builtin call operator (@).
All builtins are evaluated at compiletime.
eg: @size 2. @align Position, @trap
size
Returns the compile time size of the type.
eg: @size [] == 8, @size Position == 4
varsize
Returns the compile time size of the variable.
eg: @varsize some_str == 8
align
Returns the compile time alignment of the type.
eg: @align [] == 8, @align Position == 4
asm
Raw assembly, pasted into the generated output.
eg:
@asm {
   mov $0, %rax
}
cast
Casts a type to another.
eg: Position @cast my_var
