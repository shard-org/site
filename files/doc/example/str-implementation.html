Str Implementation

A fairly lengthy example on how an iterator implementation could be done.
:arch x86_64 linux

:struct str {
   [1:] array
   8    len
}

:CAST [1:] -> str {
   %i 8
   loop ('i ++ & #1.i = 0)
   ret {#1, i}
}

print str s:
   *write 1, s->array, s->len

entry:
   !print "Hello, World!"
   ret
