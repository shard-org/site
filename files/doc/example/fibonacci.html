Fibonacci

Here we're telling the linker to link libc.
:extern printf _ is used to define the external function printf; However as variadics are not supported, for now, it is left as undefined.

:arch x86_64 linux
:linker #LINKER -lc

:extern scanf _
:extern printf _


fibonacci 2 n, -> 2 {
   (n <= 1): ret n
   ret !fibonacci (n - 1) + !fibonacci (n - 2)
}

entry:
   %terms 2
   $scanf "%d", terms

   %i 2
   loop {'i ++}, (i < terms) {
      !fibonacci i =>> $printf "%d\n"
   }

   ret
