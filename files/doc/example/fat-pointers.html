Fat Pointers

ie. a pointer with a length field attached to it!
We use the new @cast builtin to cast a pointer to a fat pointer.
:ARCH x86_64 linux

:STRUCT str {
   [1:] array
   8    len
}

:CAST [1:] -> str {
   %i 8
   loop ('i ++ & #1.i = 0)
   ret {#1, i}
}

print str s:
   *write 1, s->array, s->len

entry:
   !print @cast "Hello, World!"
   ret
